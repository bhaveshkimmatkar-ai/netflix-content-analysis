# Netflix Content Analysis

## Tools Used
Python | Pandas | Matplotlib | Seaborn

## About
Performed Exploratory Data Analysis (EDA) on a Netflix
dataset of 8,000+ titles to uncover content trends
and distribution patterns.

## Key Insights
- Drama and International content dominates 40%+ of catalog
- Analyzed content by genre, country, and release year
- Visualized year-wise growth trends from 2000 to 2021
- Data cleaning performed on missing values and duplicates
